Program edukacyjny wirtualne laboratorium przygotowany w ramach pracy magisterskiej na kierunku
Matematyka II, Geometria i grafika komputerowa na Politechnice Gdańskiej.

Aby uruchomić program należy pobrać i wypakować pliki z archiwum "wirtualne laboratorium.rar",
a następnie włączyć aplikację "virtual laboratory".
